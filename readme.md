# Useful Lentil

A YouTube downloader UI, written in [Electron](https://www.electronjs.org/) and [ytdl-core](https://github.com/fent/node-ytdl-core/).

![](screenshots/home.png)

# Running
1. Install the dependencies: `npm i`
2. Run: `npm start`
